MeshData#

class pyqtgraph.opengl.MeshData(
vertexes=None,
faces=None,
edges=None,
vertexColors=None,
faceColors=None,
)[source]#

Class for storing and operating on 3D mesh data. May contain:

  • list of vertex locations

  • list of edges

  • list of triangles

  • colors per vertex, edge, or tri

  • normals per vertex or tri

This class handles conversion between the standard [list of vertexes, list of faces] format (suitable for use with glDrawElements) and ‘indexed’ [list of vertexes] format (suitable for use with glDrawArrays). It will automatically compute face normal vectors as well as averaged vertex normal vectors.

The class attempts to be as efficient as possible in caching conversion results and avoiding unnecessary conversions.
